Provides transformers that move between the Hibernate (HIB) and DTO layers. These classes are XSnapshot converters, and come in pairs: one class for the HIB -> DTO conversion and another for the DTO -> HIB conversion. For example, StringConverter goes from String fields in the Hibernate model to ISO St; and StConverter goes from ISO St to String fields.

Conversion Rules

When converting from HIB to DTO:
  • Converters never produce null DTOs.
  • Null input produces ISO types with NullFlavor.NI (for simple types) or empty sets (for DSets).
  • NullFlavor.NI is the old NullFlavor used.
When converting from DTO to HIB:
  • Null inputs produce null outputs.
  • NullFlavored inputs produce null outputs. Implication: we lose flavor information during this transformation and cannot recover it.
  • Some OCL constraints may be checked during conversion.

ISO Localization

The DTO classes use a localized version of the full ISO data types. Localized ISO data types use a subset of the full fields. For instance, none of our classes have the validTimeHigh or validTimeLow fields from HXIT.
