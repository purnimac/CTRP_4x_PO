Requirements:
  * Ant 1.8 or higher
  * Maven 2.2 or higher
  * Java 7
  * Postgres 9.2
  
Prior to running an install for the first time, please create a database and a user. User needs to have privileges to create database objects (tables, views, etc).
  
Modifying default properties
  * Create local.properties to overwrite any properties in install/upgrade.properties
  
Please refer to NCI Wiki for full installation instructions: https://wiki.nci.nih.gov/x/gICHCw
